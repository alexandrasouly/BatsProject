particle filtering model

We calculate conditional likelihood of SIR with no seasonal forcing at Aarons posterior parameter values (model 5).

pf <- replicate(5, pfilter(pomp_object, params = params, Np=2000))

ll <- sapply(pf,logLik)
logmeanexp(ll,se=TRUE)
                     se 
-43.8419792   0.0116115 
plot(pf[[1]])

We set IF2 to estimate the parameters with 40 iterations and 2000 particles.

foreach(start = iter(mf1)) %dopar% {
start %>%
  traces() %>%
  melt() %>%
  filter(variable %in% c("loglik","R0", "gamma_val", "omega_m_val", "zeta", "c",
                 "s", "phi", "disp", "d")) %>%
  ggplot(aes(x=iteration,y=value))+
  geom_line()+
  facet_wrap(~variable,scales="free_y")+
  guides(color=FALSE)
}
