We can beautifully see how zeta and d correlate when nothing else changes. With applying a smoother, we see the different layers in the likelihood surface. The large deep blue layer on the last plot is what we need to maximalise the likelihood. This means we will only need to fit one of zeta and d if this surface is the same for other parameter choices.

Now we need to try and see if this works the same with any other fixed params, or if we just found something special true only for one set of fixed other params. (I really hope this is universal!) If it is true, we will need to find the function that describes that layer, generate more points on and around it to narrow the region down, and were set, one fewer params to fit! Then we can think about exploring if this is true for the other models as well.
